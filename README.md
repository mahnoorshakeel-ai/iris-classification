# Iris Flower Classification

## Overview
Decision Tree Classifier to classify Iris flowers into 3 species.

## Libraries Used
- Pandas, Numpy, Seaborn, Matplotlib, Scikit-learn

## Project Steps
1. Load Dataset
2. Exploratory Data Analysis (EDA)
3. Visualization (Pairplot, Heatmap, Boxplot)
4. Decision Tree Model Training
5. Accuracy Evaluation

## Result
**Accuracy: 100%**

## Dataset
Built-in Sklearn Iris Dataset
